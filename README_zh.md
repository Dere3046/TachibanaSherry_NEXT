## 项目介绍

TachibanaSherry_NEXT 是基于原 TachibanaSherry 项目的优化升级版本，在原功能基础上进行了性能改进和功能扩展。

## 分支说明

本项目包含以下主要分支：

| 分支 | 说明 |
|------|------|
| **Android_1.0** | 📱 将原项目移植到 Android 平台，使用 Android IDE 构建（已停止维护） |
| **Android_1.1** | 🎨 在 Android 移植版基础上添加了更舒适的 UI 和调色板功能，使用 Android IDE 构建（已停止维护） |
| **Android_1.2** | 🖼️ 在 Android_1.1 基础上增加了图片预览功能，使用 Android IDE 构建（已停止维护） |
| **main** | ⌨️ 最新的 PC 版本分支，通过键盘钩子实现回车热键和 Ctrl+回车热键，使用 `python setup.py build_ext --inplace` 编译 C 扩展 |

> **注意：** Android 版本已停止维护，历史版本代码保存在各自分支内。
