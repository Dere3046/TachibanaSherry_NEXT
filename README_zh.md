# TachibanaSherry_NEXT

[English](README.md) | [中文文档](README_zh.md)

## 项目介绍

TachibanaSherry_NEXT 是基于原 TachibanaSherry 项目的优化和升级版本，在原有功能基础上进行了性能提升和功能扩展。

## 分支说明

本项目包含以下主要分支：

- **Android_1.0**: 将原项目移植到Android平台，使用Android IDE构建
- **Android_1.1**: 在Android移植基础上添加了更舒适的UI与调色板功能，使用Android IDE构建
- **main**: 基于原项目的1.2版本，通过劫持键盘实现enter热键与ctrl+enter热键，使用 `python setup.py build_ext --inplace` 编译C扩展
- **guide**: 向导分支，为主分支提供各分支功能说明
