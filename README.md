# TachibanaSherry_NEXT

[English](README.md) | [中文文档](README_zh.md)

## Project Introduction

TachibanaSherry_NEXT is an optimized and upgraded version based on the original TachibanaSherry project, with performance improvements and feature extensions built upon the original functionality.

## Branch Description

This project includes the following main branches:

- **Android_1.0**: Ported the original project to Android platform, built using Android IDE
- **Android_1.1**: Added more comfortable UI and color palette features to the Android port, built using Android IDE
- **main**: Based on version 1.2 of the original project, implements enter hotkey and ctrl+enter hotkey through keyboard hijacking, uses `python setup.py build_ext --inplace` to compile C extensions
- **guide**: Guide branch, provides functional descriptions of each branch for the main branch
